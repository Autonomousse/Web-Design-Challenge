Summary: Latitude vs Weather

The purpose of this project was to analyze how different characteristics of weather change as we approach the equator. To investigate, we first pulled data from OpenWeatherMap API and created a dataset consisting of over 500 cities scattered across the world. Next, we utilized Python and the Matplotlib module to inspect the data and create visual representations of cloudiness, humidity, max temperature, and wind speed against latitude. This site includes those visualizations accompanied by insights into potential trends and a sample dataset.
